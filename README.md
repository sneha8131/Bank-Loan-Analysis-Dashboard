Bank Personal Loan Analysis Dashboard

Project Overview

This project is an interactive Bank Personal Loan Analysis Dashboard developed using Power BI. The dashboard analyzes customer loan data to provide insights into loan approvals, customer profiles, funding performance, and key factors influencing personal loan acceptance, helping banks make informed lending decisions.

Features
Loan Approval & Rejection Analysis
Customer Demographics Insights
Income & Education-Based Analysis
Funded Amount & Repayment Analysis
Product Usage Analysis (CD Account, Credit Card, Online Banking)
KPI Tracking
Key Influencers Analysis
Interactive Filters & Visualizations
